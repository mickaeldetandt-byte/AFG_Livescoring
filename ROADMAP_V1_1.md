# Roadmap V1.1 – AFG LiveScoring

## Objectif
Améliorer l’expérience utilisateur sans casser la V1.0.0 figée.

## Améliorations envisagées
- Amélioration UX terrain mobile
- Boutons encore plus grands sur écrans de score
- Scroll / affichage ClubHouse PC
- Nettoyage visuel des tableaux
- Suppression des colonnes ID inutiles
- Amélioration export Excel
- Raffinement display / leaderboard
- Version visible dans plusieurs pages
- Ajustements mineurs après retours terrain

## Règle
Aucune modification du core scoring sans nécessité.
